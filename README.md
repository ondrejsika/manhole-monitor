### Abstract

Monitor twisted apps via manhole

### Usage



    usage: python manhole-monitor.py [-h] [--show-input] host port wait py_file

    positional arguments:
      host
      port
      wait
      py_file

    optional arguments:
      -h, --help    show this help message and exit
      --show-input
